Header CTA — visual preview

Three candidate replacements for the current “Update or verify a listing” button. Each variant is rendered inside the real ESRF overlay masthead using existing tokens (Archivo / IBM Plex Mono, cream / ink / accent), so what you see is what you’d ship.

Recommended · A “Share information” — ghost pill, current shape
ESRF.net
Foundation Directory Atlas Dispatch Share information Contribute
Why this is the recommended variant. Keeps the existing ghost-pill silhouette already established in the masthead — so the change is wording-only, not visual disruption. The label is short, neutral, and inviting: it does not imply ownership (“claim”), bureaucracy (“opgave / wijzigen informatie”), or obligation. Works equally well in both languages.
  • NL: Deel informatie
  • EN: Share information
Destinations: /submit-news?lang=nl · /submit-news?lang=en · /submit-news?lang=nl&mode=change_request
Alternative · B “Contribute or update” — solid accent pill
ESRF.net
Foundation Directory Atlas Dispatch Contribute or update Contribute
Why consider this. Same shape, but filled with the brand accent so the CTA reads first in the masthead. Higher click-affordance, stronger “primary action” signal. Risk: it sits next to the orange “Contribute” donate link, which could create a colour collision — Wouter to judge. Works well if the donate link migrates to a less saturated treatment.
  • NL: Bijdragen of wijzigen
  • EN: Contribute or update
Destinations: /submit-news?lang=nl&mode=change_request · /submit-news?lang=en&mode=change_request
Formal · C “Submit information” — ghost pill with icon
ESRF.net
Foundation Directory Atlas Dispatch Submit information Contribute
Why consider this. The most formal register — “Informatie doorgeven / Submit information” reads close to a government or institutional service. The small upload glyph clarifies that this is an action, not a destination. Best fit if ESRF wants to lean further into its “Foundation / institutional” voice. Slightly heavier than Variant A, but still well within the existing visual vocabulary.
  • NL: Informatie doorgeven
  • EN: Submit information
Destinations: /submit-news?lang=nl · /submit-news?lang=en
